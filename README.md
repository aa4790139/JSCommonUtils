# js_common_utils
前端开发：常规工具库
# 包含内容：
- ByteUtils : 基本类型与字节相关转换
- CharUtils : 字符转换相关处理
- GzipUtils : gzip 压缩 和 解压

# 安装
```
npm install js_common_utils
```

# 使用
- 浏览器
```
import {charUtils, byteUtils, gzipUtils} from 'js_common_utils'
```
- Node.js

```

var {charUtils, byteUtils, gzipUtils} = require('js_common_utils');

```
